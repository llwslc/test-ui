## 已建

- **[NOVA](nova.md)** —— 科幻霓虹 HUD / Sci-fi neon:深空黑蓝底 + 电光青霓虹 + 品红与警示红点缀;锐利全大写无衬线展示体 + 等宽数据标注;锐角 clip-path 切角面板、严格栅格、雷达准星与战术坐标;霓虹辉光、扫描线、暗角网格、玻璃微反;辉光脉动、扫描线流过、锐利滑入。

## 候选

- **SUMI** —— 水墨禅 / Zen ink:宣纸本白 + 焦墨黑 + 单枚朱印;毛笔书法标题 + 细明朝体正文;竖排、大量留白、不对称错落;墨晕飞白、宣纸纤维、印章压痕;笔锋缓显、墨滴扩散晕开。
- **ORMOLU** —— 洛可可宫廷 / Rococo:象牙白 + 鼠尾草绿 + 旧金 + 胭脂粉;高对比衬线 + 花体首字母;卷草涡纹、对称卡图什、繁复花边框;大理石纹、金箔、丝绸高光;优雅缓展、卷帘揭开。
- **NOCTURNE** —— 暗夜花园 / Dark botanical:深茄紫黑 + 暗酒红 + 黄铜金 + 骨白;优雅衬线 + 手写斜体;缠枝花卉、画框分区、对称构图;深色花卉壁纸纹、丝绒、幽微暖光;花朵缓绽、淡入浮现。
- **PULP** —— 复古廉价小说 / Pulp paperback:做旧米黄 + 砖红 + 芥末黄 + 墨青;厚重板衬线标题 + 窄身正文;厚黑边框、票券版式、贴纸印章;网点、做旧污渍黄斑、套色偏移;翻页、印章砸落。
- **TOTEM** —— 民艺图腾 / Folk craft:赭石 + 靛蓝 + 骨白 + 朱砂;木刻粗衬线 + 几何无衬线;边饰带、对称纹样、织物分块网格;木纹、扎染晕、织线纹理;节奏步进、纹样旋转。
- **PRIMARY** —— 包豪斯构成主义 / Bauhaus:暖白纸 + 三原色红黄蓝 + 纯黑;粗黑无衬线展示体 + 几何无衬线;圆、三角、方基本形、模数硬网格、对角动势;实色平涂、粗黑描边、零纹理;硬切入场、块面滑移、形旋转。
- **KILN** —— 陶土侘寂 / Wabi-sabi:窑烟黑 + 燕麦米 + 赤陶橙;人文衬线 + 柔无衬线;大留白、手感不规则边、偏心构图;粗陶釉面、纸浆纹理、柔长影;缓淡入、微沉降回弹。
- **RIOT** —— 朋克剪报 / Punk zine:复印灰白 + 荧光橙 + 死黑 + 荧光粉;勒索信拼贴字 + 粗黑无衬线;歪斜拼贴、撕边、胶带补丁、错位网格;复印噪、半调网点、墨糊、折痕;抖动、撕拉、咔哒硬切。
- **CHROME** —— 千禧液态铬 / Y2K aero:钢蓝 + 液态铬银 + 水蓝凝胶 + 全息虹彩;圆胖科技无衬线 + 拉伸标题;气泡按钮、椭圆面板、镜面反射布局;液态金属、镜头光晕、凝胶高光、扫光;果冻弹动、镜面流光。
- **BRASS** —— 蒸汽朋克 / Steampunk:黄铜金 + 紫铜橙 + 做旧皮革棕 + 暗炭底;维多利亚装饰衬线 + 机械等宽刻度;铆钉金属框、齿轮与压力表、管路连线、仪表盘版式;拉丝黄铜、铆钉、做旧划痕、铜绿锈斑、蒸汽雾;齿轮咬合转、指针摆动、阀门开合。
