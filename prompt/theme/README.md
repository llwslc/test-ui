# Themes —— kit 视觉方向

> kit 视觉方向索引。下面每条都按同一顺序、用「；」分隔写五个方面：**配色；字体；形状与布局；纹理与效果；动效**。「已建」是已落地的 kit，「候选」是还没做的方向。

## 已建

- **[NOVA](nova.md)** —— 科幻霓虹 HUD / Sci-fi neon：深空黑蓝底 + 电光青霓虹 + 品红与警示红点缀；锐利全大写无衬线展示体 + 等宽数据标注；锐角 clip-path 切角面板、严格栅格、雷达准星与战术坐标；霓虹辉光、扫描线、暗角网格、玻璃微反；辉光脉动、扫描线流过、锐利滑入。
- **[ABYSS](abyss.md)** —— 克苏鲁深渊魔典 / Eldritch grimoire：墨绿黑底 + 深渊青辉光 + 暖羊皮文字 + 琥珀点缀；魔典衬线标题 + 暖羊皮正文 + 手绘墨感；湿石板面板、睁眼符印、对称魔典版式、手绘墨线边；深渊青辉、墨晕、石板纹、符印；缓辉呼吸、墨渗、眼睛开阖。
- **[BRASS](brass.md)** —— 蒸汽朋克 / Steampunk：黄铜金 + 紫铜橙 + 铜绿 + 暖铁炭底；维多利亚展示衬线 + 工业正文衬线 + 机械等宽刻度；机加工圆角 + 双层黄铜 bevel 边框 + 角部铆钉、齿轮与压力表、仪表盘版式；拉丝黄铜、铆钉、铜绿锈斑、蒸汽雾、陈旧金属颗粒；齿轮咬合转、指针摆动、阀门落位。
- **[PRISM](bauhaus.md)** —— 包豪斯构成 / Bauhaus：暖纸本白 + 三原色红黄蓝 + 纯黑；粗黑无衬线展示体 + 几何无衬线正文 + 等宽数据刻度；锐角矩形与正圆基本形、单层粗黑描边框、模数硬网格加对角动势；实色平涂、零纹理、零辉光、硬切偏移实影；硬切入场、块面滑移、形旋转。
- **[RIOT](riot.md)** —— 朋克剪报 / Punk zine：复印灰白 + 荧光橙 + 死黑 + 荧光粉；勒索信拼贴字 + 粗黑无衬线；歪斜拼贴、撕边、胶带补丁、错位网格；复印噪、半调网点、墨糊、折痕；抖动、撕拉、咔哒硬切。
- **[HANABI](hanabi.md)** —— 二次元赛璐璐 / Anime cel：淡朝空底 + 应援粉 + 水色青 + 星光金 + 冷紫蓝阶影；日文海报粗黑展示体 + 像素注记体；圆角赛璐璐块、斜切铭牌、四角锁定框、分格量表；墨蓝描线、平涂填色、硬边彩影、高光白条、半调网点；果冻弹落位、斜纹流动、✦ 火花明灭、刀光按压。

## 候选

- **SUMI** —— 水墨禅 / Zen ink：宣纸本白 + 焦墨黑 + 单枚朱印；毛笔书法标题 + 细明朝体正文；竖排、大量留白、不对称错落；墨晕飞白、宣纸纤维、印章压痕；笔锋缓显、墨滴扩散晕开。
- **ORMOLU** —— 洛可可宫廷 / Rococo：象牙白 + 鼠尾草绿 + 旧金 + 胭脂粉；高对比衬线 + 花体首字母；卷草涡纹、对称卡图什、繁复花边框；大理石纹、金箔、丝绸高光；优雅缓展、卷帘揭开。
- **NOCTURNE** —— 暗夜花园 / Dark botanical：深茄紫黑 + 暗酒红 + 黄铜金 + 骨白；优雅衬线 + 手写斜体；缠枝花卉、画框分区、对称构图；深色花卉壁纸纹、丝绒、幽微暖光；花朵缓绽、淡入浮现。
- **PULP** —— 复古廉价小说 / Pulp paperback：做旧米黄 + 砖红 + 芥末黄 + 墨青；厚重板衬线标题 + 窄身正文；厚黑边框、票券版式、贴纸印章；网点、做旧污渍黄斑、套色偏移；翻页、印章砸落。
- **TOTEM** —— 民艺图腾 / Folk craft：赭石 + 靛蓝 + 骨白 + 朱砂；木刻粗衬线 + 几何无衬线；边饰带、对称纹样、织物分块网格；木纹、扎染晕、织线纹理；节奏步进、纹样旋转。
- **KILN** —— 陶土侘寂 / Wabi-sabi：窑烟黑 + 燕麦米 + 赤陶橙；人文衬线 + 柔无衬线；大留白、手感不规则边、偏心构图；粗陶釉面、纸浆纹理、柔长影；缓淡入、微沉降回弹。
- **CHROME** —— 千禧液态铬 / Y2K aero：钢蓝 + 液态铬银 + 水蓝凝胶 + 全息虹彩；圆胖科技无衬线 + 拉伸标题；气泡按钮、椭圆面板、镜面反射布局；液态金属、镜头光晕、凝胶高光、扫光；果冻弹动、镜面流光。
- **ARCADE** —— 8-bit 街机 / Retro arcade：机柜黑 + 荧光蓝红黄青鬼影色 + 磷光白；像素点阵展示体 + 等宽计分体；砖砌像素边框、分段血槽、HI-SCORE 计分牌、▶ 选择光标、摇杆键位图记、迷宫点阵走线；CRT 弧面暗角、荫罩栅格纹、金币樱桃精灵点缀、像素锯齿边；PRESS START 闪烁、逐帧步进、吃点连珠、方块硬降落位。
- **BITMAP** —— 1-bit 位图 / One-bit dither：纸白 + 纯墨黑双色；粗黑像素系统体 + 点阵正文；斑马纹标题栏、□ 关闭钮、单像素阶梯圆角、反白选中行、方钮滚动条、下拉阴影错一格；有序抖动网纹充当灰阶、50% 棋盘格填充、软盘沙漏像素图记、零反锯齿；蚂蚁线选框爬行、窗格硬弹开、光标手指点按。
- **MOCHI** —— 软陶泡芙 / Claymorphic squish：奶油白底 + 马卡龙粉蓝黄 + 可可深棕字；圆胖几何无衬线；胖墩药丸钮、挤花描边、指压凹窝、咬缺角标、旗签小标签、全程无锐角；内外双层软影塑充气凸面、哑光陶面、糖针撒点、糖霜滴边；挤压回弹、戳下瘪一口气、蒸汽袅袅。
- **SCRIBBLE** —— 手账涂鸦 / Notebook doodle：横线本纸白 + 圆珠蓝 + 荧光笔黄粉；手写体标题 + 圆润正文；线圈本页边、便利贴块、回形针夹角、手画不闭合框、手绘箭头批注、页角折耳；铅笔灰、纸纹、荧光笔涂重点、咖啡渍圈、涂改带白块；线条手绘渐显、波浪线爬行、贴纸拍落。
- **AXON** —— 轴测蓝图 / Isometric blueprint：蓝图青灰底 + 制图白线 + 单枚橙强调；窄身几何无衬线 + mono 尺寸标注；30° 轴测投影、挤出体面板、图纸标题栏、尺寸标注箭头、引出线泡注、爆炸图层叠；坐标纸网格、剖切线、比例尺与指北针、长条投影；沿轴滑入、层层抬升、标注线延展点亮。
- **CORONA** —— 日蚀玻璃 / Eclipse glass：黑曜石底 + 日冕琥珀金 + 大气青；几何科技展示体 + 宽距大写标注；食相盘与遮掩月影、轨道刻度环、贝利珠点串、观测坐标注记、毛玻璃面板、径向渐晕；backdrop 模糊、1px 白内描、有色光晕影、日冕丝流；金环闪现、食分推进、辉光缓呼吸。
- **RELIQUARY** —— 未来圣骸 / Future reliquary：黑曜石 + 骨瓷白 + 氧化金 + 毒液信号绿 + 动脉警报红；碑铭高反差衬线展示体 + 工程窄体无衬线 + 档案等宽体；活体圣骸中轴构图、甲壳切面、肋骨网格、虹膜核心、机械椎骨开关、轨道坐标与收容舱框；虹彩金属甲壳、骨瓷断面、氧化金缝线、生物荧光神经、玻璃标本层与未来考古噪点；甲壳解封、虹膜旋开、神经脉冲、图层剖解抬升、收容结构闭锁与衰变碎落。
- **GAZETTE** —— 报章排印 / Broadsheet：新闻纸白 + 油墨黑 + 一色墨蓝点缀；高对比展示衬线大标 + 正文衬线 + mono 眉题；哥特体报头、期号日期眉线、多栏发丝分栏、首字下沉、引语拉出大字、十字对版标；新闻纸纹、套印微移、铅字压痕；近乎静止、hover 亮下划线、油墨未干微晕。
