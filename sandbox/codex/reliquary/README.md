# RELIQUARY / Future Relic System

独立实验主题：未来考古博物馆 × 生物机械 × 高级钟表。

## 体验结构

- Artifact：可解除封存的多层 SVG 圣骸
- Anatomy：甲壳、骨架、神经和核心四层交互剖面
- Component Lab：按钮、表单、导航、反馈、数据与 Dialog
- Motion Grammar：Awaken、Lock、Pulse、Decay 四类语义动效
- Material Archive：黑曜石、骨瓷、氧化金、毒液信号与警报红

页面还包含 Canvas 粒子场、鼠标视差、滚动进度、可选 Web Audio 合成反馈和响应式布局。纯静态实现，无外部依赖。
